# 参数调整指南

本文档提供详细的参数调整方法，帮助您自定义机械臂运动行为。

---

## 📍 目标位置参数

### 位置坐标修改

**文件**: `main.py`  
**位置**: 约第95-96行

```python
# 原始值
target_A_pos = [-0.13, 0.5, 0.1]   # A位置：接近立方体
target_B_pos = [-0.13, 0.65, 0.1]  # B位置：推动立方体

# 示例修改
target_A_pos = [-0.13, 0.45, 0.12]  # 更早接近，高度提升2cm
target_B_pos = [-0.13, 0.70, 0.12]  # 推得更远
```

**坐标含义**：
- **X**: 左右位置（负值=左侧，正值=右侧）
- **Y**: 前后位置（正值=向前）
- **Z**: 上下位置（正值=向上）

**安全范围**（UR5e工作空间）：
- X: -0.8 ~ 0.8 米
- Y: -0.8 ~ 0.8 米
- Z: 0.0 ~ 1.2 米

## 🎯 末端姿态参数

### 欧拉角修改

**文件**: `main.py`  
**位置**: 约第97行

```python
# 原始值（欧拉角，单位：弧度）
ee_euler = [3.14, 0, 1.57]  # [roll, pitch, yaw]

# 示例修改
ee_euler = [3.14, 0.5, 1.57]  # 增加俯仰角
ee_euler = [3.14, 0, 0]       # 改变水平朝向
```

**角度含义**：
- **Roll (绕X轴)**: 3.14弧度 = 180° (末端翻转)
- **Pitch (绕Y轴)**: 0 = 水平
- **Yaw (绕Z轴)**: 1.57弧度 = 90° (基座旋转)

**单位转换**：
```python
import math
度数 = 弧度 * 180 / math.pi
弧度 = 度数 * math.pi / 180
```

## ⏱️ 运动速度参数

### 轨迹步数调整

**文件**: `main.py`  
**位置**: 约第109行和126行

```python
# 原始值
trajectory_to_A = JointSpaceTrajectory(start_joints, target_A_joints, steps=100)
trajectory_to_B = JointSpaceTrajectory(data.qpos[:6], target_B_joints, steps=150)

# 调整运动速度
# 步数越大 → 运动越慢 → 越平滑
trajectory_to_A = JointSpaceTrajectory(start_joints, target_A_joints, steps=200)  # 变慢2倍
trajectory_to_B = JointSpaceTrajectory(data.qpos[:6], target_B_joints, steps=50)   # 变快3倍
```

**时间计算**：
- 时间步长 = 0.002秒（在scene.xml中定义）
- 总时间 = 步数 × 0.002秒
- 100步 = 0.2秒
- 150步 = 0.3秒

## 🎯 到达精度参数

### 容差调整

**文件**: `main.py`  
**位置**: 约第121行和131行

```python
# 原始值（容差：0.02弧度 ≈ 1.15°）
if np.allclose(data.qpos[:6], target_A_joints, atol=0.02):

# 调整精度
# 容差越小 → 越精确 → 可能更难到达
if np.allclose(data.qpos[:6], target_A_joints, atol=0.01):  # 更严格（0.57°）
if np.allclose(data.qpos[:6], target_A_joints, atol=0.05):  # 更宽松（2.86°）
```

**建议值**：
- 高精度任务: 0.01 弧度
- 一般任务: 0.02 弧度（默认）
- 快速测试: 0.05 弧度


## 🎥 相机视角参数

### 初始视角调整

**文件**: `main.py`  
**位置**: `viewer_init()` 函数

```python
def viewer_init(viewer):
    viewer.cam.type = mujoco.mjtCamera.mjCAMERA_FREE  # 自由视角
    viewer.cam.lookat[:] = [0, 0.5, 0.5]   # 注视点（XYZ）
    viewer.cam.distance = 2.5               # 相机距离
    viewer.cam.azimuth = 180                # 方位角（水平旋转）
    viewer.cam.elevation = -30              # 仰角（俯视角度）

# 示例修改：俯视视角
def viewer_init(viewer):
    viewer.cam.lookat[:] = [0, 0.5, 0.3]   # 注视点降低
    viewer.cam.distance = 1.5               # 拉近
    viewer.cam.azimuth = 180
    viewer.cam.elevation = -60              # 更俯视
```

**参数说明**：
- **lookat**: 相机注视的中心点坐标
- **distance**: 相机距离注视点的距离（米）
- **azimuth**: 水平旋转角度（0-360°）
- **elevation**: 垂直角度（-90°=俯视，90°=仰视）

## 🔋 力传感器参数

### 滑动窗口大小

**文件**: `main.py`  
**位置**: 约第141行

```python
# 原始值（100帧滑动平均）
force_sensor = ForceSensor(model, data, window_size=100)

# 调整滤波效果
force_sensor = ForceSensor(model, data, window_size=50)   # 响应更快，噪声更大
force_sensor = ForceSensor(model, data, window_size=200)  # 更平滑，响应更慢
```

**时间换算**：
- 窗口大小100 = 0.2秒历史数据（假设500Hz）
- 窗口越大 → 越平滑 → 延迟越大

### 可视化更新频率

**文件**: `main.py`  
**位置**: `ForcePlotter` 类初始化

## ⚙️ 物理仿真参数

### 时间步长

**文件**: `model/universal_robots_ur5e/scene.xml`  
**位置**: `<option>` 标签内

```xml
<!-- 默认在ur5e.xml中定义 -->
<option timestep="0.002"/>  <!-- 2毫秒/步 = 500Hz -->

<!-- 修改仿真精度 -->
<option timestep="0.001"/>  <!-- 更精确，更慢 -->
<option timestep="0.005"/>  <!-- 更快，精度降低 -->
```

### 控制器增益

**文件**: `model/universal_robots_ur5e/ur5e.xml`  
**位置**: `<default>` 标签内的 `<general>` 配置

```xml
<!-- 原始值（位置PD控制器）-->
<general 
    gainprm="2000"              <!-- P增益（位置）-->
    biasprm="0 -2000 -400"      <!-- D增益（速度）-->
    forcerange="-150 150"/>     <!-- 力矩限制 -->

<!-- 调整响应速度 -->
<!-- 增大P增益 → 响应更快 → 可能震荡 -->
<general gainprm="3000" ... />

<!-- 增大D增益 → 增加阻尼 → 减少震荡 -->
<general biasprm="0 -3000 -600" ... />
```

## 📊 参数调整工作流

### 推荐步骤

1. **备份原始文件**
   ```bash
   cp main.py main.py.backup
   cp model/universal_robots_ur5e/scene.xml scene.xml.backup
   ```

2. **小步修改**
   - 每次只修改一个参数
   - 立即测试观察效果

3. **记录修改**
   - 在代码中添加注释
   - 记录哪些参数组合效果好

4. **验证安全性**
   - 确保目标位置在工作空间内
   - 检查关节角度在限位范围内
