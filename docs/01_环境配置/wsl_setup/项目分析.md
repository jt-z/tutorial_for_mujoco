# MuJoCo 仿真实战教程项目分析

## 项目概述

这是一个基于 MuJoCo 物理引擎的机械臂仿真教程项目，配合 CSDN 博客文章使用。项目演示了如何使用 MuJoCo 控制 UR5e 机械臂完成抓取任务，并实时可视化接触力数据。

## 核心功能

### 1. 机械臂运动控制
- 使用 **UR5e** 工业机械臂模型（Universal Robots 的 6 自由度协作机器人）
- 实现了**逆运动学（IK）**求解：通过 `ikpy` 库将末端执行器的目标位姿（位置 + 姿态）转换为关节角度
- **关节空间轨迹规划**：在关节空间进行线性插值，生成平滑的运动轨迹

### 2. 力传感器与数据处理
- 集成力传感器监测接触力
- 实现**滑动平均滤波器**（窗口大小 100）来平滑传感器噪声
- 实时采集并处理三维力向量数据

### 3. 可视化系统
- **MuJoCo 3D 渲染**：实时显示机械臂和环境的物理仿真
- **力向量可视化**：使用 matplotlib 3D 图表实时显示：
  - 力的方向（红色主箭头 + 蓝色箭头）
  - XY 和 XZ 平面投影（虚线）
  - 力大小指示条（青色竖线）
  - 力数值显示

## 技术架构

### 核心类设计

#### 1. `ForceSensor` - 力传感器数据处理
```python
class ForceSensor:
    def __init__(self, model, data, window_size=100)
    def filter(self)  # 返回滤波后的三维力向量
```
- 读取 MuJoCo 传感器数据（`data.sensordata[:3]`）
- 使用 `deque` 实现滑动窗口滤波降噪
- 对力向量取反（`* -1`）以匹配坐标系

#### 2. `ForcePlotter` - 实时力可视化
```python
class ForcePlotter:
    def __init__(self, update_interval=20)
    def plot_force_vector(self, force_vector)
```
- 3D 向量场绘制（使用 matplotlib 的 3D axes）
- 按帧更新（每 20 帧更新一次，减少计算开销）
- 自动归一化力方向向量

#### 3. `JointSpaceTrajectory` - 关节轨迹生成器
```python
class JointSpaceTrajectory:
    def __init__(self, start_joints, end_joints, steps)
    def get_next_waypoint(self, qpos)
```
- 线性插值生成路径点
- 自动检测到达条件（`np.allclose(atol=0.02)`）并更新目标点
- 使用生成器模式节省内存

#### 4. `viewer_init()` - 渲染器初始化
```python
def viewer_init(viewer):
    viewer.cam.type = mujoco.mjtCamera.mjCAMERA_FREE
    viewer.cam.lookat[:] = [0, 0.5, 0.5]
    viewer.cam.distance = 2.5
    viewer.cam.azimuth = 180
    viewer.cam.elevation = -30
```
- 设置自由视角相机
- 配置观察点、距离和角度参数

## 运动任务详解

### 关节角度配置
```python
start_joints = [-1.57, -1.34, 2.65, -1.3, 1.55, 0]  # 初始关节角（弧度）
```
对应末端执行器位姿：
- 位置：`[-0.13, 0.3, 0.1]`
- 欧拉角：`[3.14, 0, 1.57]`（翻转 180°，绕 Z 轴 90°）

### 目标位姿
```python
ee_pos = [-0.13, 0.6, 0.1]        # 末端位置
ee_euler = [3.14, 0, 1.57]        # 末端姿态（欧拉角）
```

### 运动描述
机械臂末端从 `y=0.3` 移动到 `y=0.6`（向前移动 **0.3m**），朝向红色立方体进行接触。

### 逆运动学求解
```python
ref_pos = [0, 0, -1.57, -1.34, 2.65, -1.3, 1.55, 0, 0]  # 参考位置
ee_orientation = tf.euler.euler2mat(*ee_euler)          # 转换为旋转矩阵
joint_angles = my_chain.inverse_kinematics(
    ee_pos, ee_orientation, "all", initial_position=ref_pos
)
end_joints = joint_angles[2:-1]  # 提取 6 个关节角度
```

## 控制流程

### 主循环
```python
with mujoco.viewer.launch_passive(model, data) as viewer:
    viewer_init(viewer)
    while viewer.is_running():
        # 1. 获取下一个路径点
        waypoint = joint_trajectory.get_next_waypoint(data.qpos[:6])
        
        # 2. 设置控制指令
        data.ctrl[:6] = waypoint
        
        # 3. 读取并滤波传感器数据
        filtered_force = force_sensor.filter()
        
        # 4. 可视化力向量
        force_plotter.plot_force_vector(filtered_force)
        
        # 5. 物理仿真步进
        mujoco.mj_step(model, data)
        
        # 6. 同步渲染器
        viewer.sync()
```

### 关键特性

1. **闭环控制**
   - 实时读取当前关节位置 `data.qpos[:6]`
   - 动态判断是否到达当前路径点
   - 自动更新下一个目标点

2. **容差控制**
   ```python
   np.allclose(qpos, self.waypoint, atol=0.02)
   ```
   - 绝对误差容差：0.02 弧度（约 1.15°）

3. **数据滤波**
   - 滑动窗口大小：100 帧
   - 算法：简单移动平均（SMA）
   - 目的：消除传感器噪声

4. **性能优化**
   - 力可视化每 20 帧更新一次
   - 避免 matplotlib 渲染成为瓶颈
   - 使用 `plt.ion()` 交互模式
