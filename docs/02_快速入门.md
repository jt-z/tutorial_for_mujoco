## 运动过程说明

### 阶段1：接近立方体（约2秒）
- 机械臂末端从初始位置向**绿色球**移动
- 移动距离：20cm（沿Y轴）
- 目的：接近红色立方体但不接触

### 阶段2：推动立方体（约3秒）
- 机械臂末端继续向**蓝色球**移动
- 移动距离：15cm（沿Y轴）
- 效果：推动红色立方体向前移动


```python
target_A_pos = [-0.13, 0.5, 0.1]   # A位置：接近立方体
```

## 常见问题

### Q1: 窗口打开后看不到机械臂

**解决方法：**
- 尝试用鼠标滚轮缩小视角
- 或按住鼠标左键拖拽旋转视角

### Q2: 程序报错找不到文件

**检查事项：**
```bash
# 确认当前目录
pwd
# 应该显示：/home/zjt/dev/On_Git_Projects/tutorial_for_mujoco

# 检查文件是否存在
ls model/universal_robots_ur5e/scene.xml
```

### Q3: 机械臂不动或卡住

**可能原因：**
- 目标位置超出工作空间
- 逆运动学求解失败

**解决方法：**
- 恢复原始目标位置值
- 查看[故障排除文档](./06_参考手册/故障排除.md)

### Q4: WSL环境下窗口无法显示

**解决方法：**
```bash
# 使用提供的脚本运行
cd docs/01_环境配置
./run_with_wslg.sh
```
