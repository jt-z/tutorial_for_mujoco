#### 两阶段运动控制
- 实现从单阶段到两阶段运动的升级
- 阶段1: 接近立方体（A位置）
- 阶段2: 推动立方体（B位置）
- 状态机管理：`to_A` → `to_B` → `completed`
- 实时控制台状态输出

- 添加A位置标记（绿色半透明球体）

## [1.0.0] - 2026-07-20 (初始版本)

### ✨ 核心功能

#### 基础机械臂控制
- UR5e 6自由度机械臂仿真
- 逆运动学求解（ikpy）
- 关节空间轨迹规划
- 位置PD控制

#### 力传感器
- 末端三维力传感器
- 滑动窗口滤波（100帧）
- 实时力向量3D可视化

#### 可视化
- MuJoCo 3D仿真窗口
- Matplotlib力向量图表
- 世界坐标系标记
- 自定义相机视角

#### 场景元素
- UR5e机械臂模型
- 红色立方体（可移动）
- 地面和光照
- 自由度关节

### 🛠️ 技术栈

- **物理引擎**: MuJoCo 3.3.4
- **运动学**: ikpy 3.3+
- **坐标变换**: transforms3d 0.4+
- **数值计算**: NumPy 1.21+
- **可视化**: Matplotlib 3.5+

### 📂 项目结构


## 版本说明

### 版本号规则

采用语义化版本号：`主版本.次版本.修订号`

- **主版本**: 重大架构变更或不兼容更新
- **次版本**: 新功能添加，向后兼容
- **修订号**: Bug修复和小改进

### 版本标签

```bash
# 查看所有版本标签
git tag

# 查看特定版本
git show v2.0.0
```
